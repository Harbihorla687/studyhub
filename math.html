Example Questions

1. Simplify: 2x + 3x

= 5x

2. Expand: (x + 2)(x + 5)

= x² + 7x + 10

3. Factorize: x² + 6x + 8

= (x + 2)(x + 4)

4. Simplify: 4a - 2a + 7

= 2a + 7

5. Expand: (x - 3)(x - 2)

= x² - 5x + 6

6. Factorize: x² - 9

= (x - 3)(x + 3)

7. Simplify: 3(x + 4)

= 3x + 12

8. Expand: (2x + 1)(x + 3)

= 2x² + 7x + 3

9. Factorize: x² + x - 6

= (x + 3)(x - 2)

10. Simplify: 5x - 2x + 9

= 3x + 9
